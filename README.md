# Gradient Descent Visualization

Interactive implementation of gradient descent 
for f(x,y) = y·sin(x) − x·cos(y).

## Features
- Symbolic gradient computation (SymPy)
- Interactive parameter tuning (ipywidgets)
- Momentum variant
- Animation + 3D surface plot

## Technologies
Python, NumPy, SymPy, Matplotlib, ipywidgets
